# Course Companion – GPT Chatbot

A student-facing chatbot for the Information Systems Business Concepts course. Uses OpenAI's GPT API to answer questions with a teaching assistant tone. Designed for deployment on Render.

## Deployment Steps

1. Push this folder to a GitHub repo
2. Create a new Web Service on [https://render.com](https://render.com)
3. Use the following settings:
   - Build command: `npm install`
   - Start command: `npm start`
   - Environment Variable: `OPENAI_API_KEY`

4. Share your Render deployment URL in Blackboard as a Web Link.

**Note**: This tool is for study support only. Do not use it for exams or graded work.
